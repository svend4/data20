1. Singleton (–û–¥–∏–Ω–æ—á–∫–∞)

–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ: –ì–∞—Ä–∞–Ω—Ç–∏—Ä–æ–≤–∞—Ç—å, —á—Ç–æ —É –∫–ª–∞—Å—Å–∞ –µ—Å—Ç—å —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω —ç–∫–∑–µ–º–ø–ª—è—Ä, –∏ –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–∏—Ç—å –≥–ª–æ–±–∞–ª—å–Ω—É—é —Ç–æ—á–∫—É –¥–æ—Å—Ç—É–ø–∞ –∫ –Ω–µ–º—É.

2. Factory Method (–§–∞–±—Ä–∏—á–Ω—ã–π –º–µ—Ç–æ–¥)

–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ: –û–ø—Ä–µ–¥–µ–ª–∏—Ç—å –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –æ–±—ä–µ–∫—Ç–∞, –Ω–æ –æ—Å—Ç–∞–≤–∏—Ç—å –ø–æ–¥–∫–ª–∞—Å—Å–∞–º —Ä–µ—à–µ–Ω–∏–µ –æ —Ç–æ–º, –∫–∞–∫–æ–π –∫–ª–∞—Å—Å –∏–Ω—Å—Ç–∞–Ω—Ü–∏—Ä–æ–≤–∞—Ç—å.

–†–µ–∞–ª–∏–∑–∞—Ü–∏—è: ```python from abc import ABC, abstractmethod

class PDFDocument(Document): def open(self): return "Opening PDF document"

–ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ

–°–æ–≤—Ä–µ–º–µ–Ω–Ω–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º dict: ```python class DocumentFactory: _creators = { "pdf": PDFDocument, "word": WordDocument, }

@classmethod def register(cls, name: str, creator): cls._creators[name] = creator

–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ: –û—Ç–¥–µ–ª–∏—Ç—å –∫–æ–Ω—Å—Ç—Ä—É–∏—Ä–æ–≤–∞–Ω–∏–µ —Å–ª–æ–∂–Ω–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞ –æ—Ç –µ–≥–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è.

4. Decorator (–î–µ–∫–æ—Ä–∞—Ç–æ—Ä)

7. Observer (–ù–∞–±–ª—é–¥–∞—Ç–µ–ª—å)

–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ: –û–ø—Ä–µ–¥–µ–ª–∏—Ç—å –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç—å –æ–¥–∏–Ω-–∫–æ-–º–Ω–æ–≥–∏–º –º–µ–∂–¥—É –æ–±—ä–µ–∫—Ç–∞–º–∏ —Ç–∞–∫, —á—Ç–æ –ø—Ä–∏ –∏–∑–º–µ–Ω–µ–Ω–∏–∏ —Å–æ—Å—Ç–æ—è–Ω–∏—è –æ–¥–Ω–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞ –≤—Å–µ –∑–∞–≤–∏—Å—è—â–∏–µ –æ—Ç –Ω–µ–≥–æ –æ–ø–æ–≤–µ—â–∞—é—Ç—Å—è –æ–± —ç—Ç–æ–º.

–†–µ–∞–ª–∏–∑–∞—Ü–∏—è: ```python from abc import ABC, abstractmethod from typing import List

class Observer(ABC): @abstractmethod def update(self, message: str): pass

class Subject: def __init__(self): self._observers: List[Observer] = []

def attach(self, observer: Observer): self._observers.append(observer)

def detach(self, observer: Observer): self._observers.remove(observer)

def notify(self, message: str): for observer in self._observers: observer.update(message)

class SMSNotifier(Observer): def update(self, message: str): print(f"SMS: {message}")

8. Strategy (–°—Ç—Ä–∞—Ç–µ–≥–∏—è)

–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ: –û–ø—Ä–µ–¥–µ–ª–∏—Ç—å —Å–µ–º–µ–π—Å—Ç–≤–æ –∞–ª–≥–æ—Ä–∏—Ç–º–æ–≤, –∏–Ω–∫–∞–ø—Å—É–ª–∏—Ä–æ–≤–∞—Ç—å –∫–∞–∂–¥—ã–π –∏ —Å–¥–µ–ª–∞—Ç—å –∏—Ö –≤–∑–∞–∏–º–æ–∑–∞–º–µ–Ω—è–µ–º—ã–º–∏.

–ö–ª–∞—Å—Å–∏—á–µ—Å–∫–∞—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è: ```python from abc import ABC, abstractmethod

class QuickSort(SortStrategy): def sort(self, data: list) -> list: print("Using QuickSort") return sorted(data) # —É–ø—Ä–æ—â–µ–Ω–Ω–æ

9. Command (–ö–æ–º–∞–Ω–¥–∞)

–ù–∞–∑–Ω–∞—á–µ–Ω–∏–µ: –ò–Ω–∫–∞–ø—Å—É–ª–∏—Ä–æ–≤–∞—Ç—å –∑–∞–ø—Ä–æ—Å –∫–∞–∫ –æ–±—ä–µ–∫—Ç, –ø–æ–∑–≤–æ–ª—è—è –ø–∞—Ä–∞–º–µ—Ç—Ä–∏–∑–æ–≤–∞—Ç—å –∫–ª–∏–µ–Ω—Ç–æ–≤ —Å —Ä–∞–∑–ª–∏—á–Ω—ã–º–∏ –∑–∞–ø—Ä–æ—Å–∞–º–∏, —Å—Ç–∞–≤–∏—Ç—å –∑–∞–ø—Ä–æ—Å—ã –≤ –æ—á–µ—Ä–µ–¥—å, –ª–æ–≥–∏—Ä–æ–≤–∞—Ç—å –∏—Ö –∏ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—Ç—å –æ—Ç–º–µ–Ω—É –æ–ø–µ—Ä–∞—Ü–∏–π.

–†–µ–∞–ª–∏–∑–∞—Ü–∏—è: ```python from abc import ABC, abstractmethod from typing import List

class Command(ABC): @abstractmethod def execute(self): pass

@abstractmethod def undo(self): pass

def execute(self): self.light.on()

def undo(self): self.light.off()

class LightOffCommand(Command): def __init__(self, light: Light): self.light = light

def execute(self): self.light.off()

def undo(self): self.light.on()

class RemoteControl: def __init__(self): self.history: List[Command] = []
